## Overview

The E-Commerce Sales Performance Dashboard is an end-to-end data analytics project that analyzes e-commerce sales data to understand sales trends, customer behavior, and product performance. The project covers data cleaning, exploratory data analysis (EDA), and interactive dashboard development using Power BI.

---

## 🎯 Objective

Build an analytics dashboard that tracks:

* Sales Performance
* Customer Behavior
* Product Performance

using an e-commerce dataset.

---

## 🛠️ Tools and Technologies

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Power BI
* GitHub

---

## 📂 Project Workflow

### 1. Data Cleaning and Preprocessing

* Handling missing values
* Removing duplicate records
* Data type conversion
* Outlier detection and removal
* Revenue calculation

### 2. Exploratory Data Analysis (EDA)

* Monthly Sales Analysis
* Product Performance Analysis
* Customer Analysis
* Country-wise Sales Analysis
* Revenue Distribution

### 3. Dashboard Development

* KPI Cards
* Monthly Sales Trend
* Top Products Analysis
* Country-wise Sales Analysis
* Customer Analysis
* Interactive Slicers

---
